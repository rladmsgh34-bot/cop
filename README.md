## 프로젝트 개요
이 저장소는 `uni34.duckdns.org` 도메인과 GCP 환경에 구축된 **n8n + Dify 기반 GitHub 영향도 및 품질 자동화 시스템** 프로젝트입니다.

- GitHub Push 이벤트 기반 상호 시스템 연동.
- 정적 코드 분석 도구(SonarQube)를 활용한 품질 개선.
